### 2.4 セクション②：基準価額推移グラフ（折れ線グラフ）

#### グラフ仕様

| 項目 | 仕様 |
|------|------|
| グラフ種別 | 折れ線グラフ（2系列） |
| グラフサイズ | 横幅：15列分、縦幅：16行分 |
| 横軸 | 対象月の営業日（日付表示、斜め45度回転） |
| 縦軸（左） | 外貨建てNAV（系列1） |
| 縦軸（右） | 円建てNAV（系列2） |
| 系列1 | 線色：青（#4472C4）、マーカー：丸 |
| 系列2 | 線色：赤（#ED7D31）、マーカー：三角 |
| 凡例 | グラフ下部に表示 |
| タイトル | 「基準価額推移（{対象年月}）」 |

#### データ取得SQL（概要）

```sql
SELECT
    NAV_DATE,
    NAV_PER_UNIT,
    NAV_JPY
FROM T_NAV_HISTORY
WHERE ISIN = @isin
  AND NAV_DATE >= @monthStart
  AND NAV_DATE <= @monthEnd
  AND NAV_JPY IS NOT NULL
ORDER BY NAV_DATE
```

### 2.5 セクション③：純資産総額推移グラフ（棒グラフ）

#### グラフ仕様

| 項目 | 仕様 |
|------|------|
| グラフ種別 | 縦棒グラフ（1系列） |
| グラフサイズ | 横幅：15列分、縦幅：16行分 |
| 横軸 | 月（直近12か月、YYYY/MM 形式） |
| 縦軸 | 純資産総額（外貨建て） |
| 棒の色 | 青（#4472C4） |
| 数値書式 | 3桁カンマ区切り |
| タイトル | 「純資産総額推移（直近12か月）」 |

#### データ取得

各月の月末日のNAVレコードから TOTAL_NET_ASSETS を取得する。月末日は「各月の最後のNAV_DATE」として取得する。

```sql
SELECT
    FORMAT(NAV_DATE, 'yyyy/MM') AS MONTH_LABEL,
    TOTAL_NET_ASSETS
FROM T_NAV_HISTORY
WHERE ISIN = @isin
  AND NAV_DATE IN (
      SELECT MAX(NAV_DATE)
      FROM T_NAV_HISTORY
      WHERE ISIN = @isin
        AND NAV_DATE >= @twelveMonthsAgo
        AND NAV_DATE <= @monthEnd
      GROUP BY FORMAT(NAV_DATE, 'yyyy-MM')
  )
ORDER BY NAV_DATE
```

### 2.6 セクション④：リターン表

#### レイアウト

```
+------------------+-------------------+-------------------+
|    Period         | Return (FCY)      | Return (JPY)      |
+------------------+-------------------+-------------------+
| Monthly           |          +1.23 %  |          +0.85 %  |
| Year-to-Date      |          +5.67 %  |          +3.45 %  |
| Since Inception   |         +23.40 %  |         +18.90 %  |
+------------------+-------------------+-------------------+
```

#### 計算仕様

| 期間 | 計算式 |
|------|--------|
| 月間リターン | (当月末NAV - 前月末NAV) ÷ 前月末NAV × 100 |
| 年初来リターン | (当月末NAV - 前年末NAV) ÷ 前年末NAV × 100 |
| 設定来リターン | (当月末NAV - 設定時NAV) ÷ 設定時NAV × 100 |

**NAV値の定義**

| 項目 | 定義 |
|------|------|
| 当月末NAV | 対象月の最終NAV_DATEのNAV_PER_UNIT（外貨）/ NAV_JPY（円） |
| 前月末NAV | 対象月の前月の最終NAV_DATEのNAV値 |
| 前年末NAV | 対象年の前年12月の最終NAV_DATEのNAV値 |
| 設定時NAV | M_FUND.INCEPTION_DATE 以降で最初のNAV_DATEのNAV値 |

**表示書式**

| 項目 | 書式 |
|------|------|
| リターン値 | 小数第2位まで表示（第3位を四捨五入） |
| 正のリターン | 「+X.XX %」、フォント色：青 |
| 負のリターン | 「-X.XX %」、フォント色：赤 |
| 計算不可時 | 「N/A」と表示（前月末NAV等が存在しない場合） |

#### Excel書式

| セル | 書式 |
|------|------|
| ヘッダ行 | 背景色：濃い青（#4472C4）、フォント色：白、太字 |
| データ行 | 交互行の背景色：薄い青（#D9E2F3） |
| リターン値 | 右揃え |

### 2.7 セクション⑤：適用為替レート一覧

#### レイアウト

```
+------------+-----------+-----------+-----------+
|    Date    |    TTM    |    TTS    |    TTB    |
+------------+-----------+-----------+-----------+
| 2025/01/06 |  149.5000 |  150.5000 |  148.5000 |
| 2025/01/07 |  149.8000 |  150.8000 |  148.8000 |
| ...        |           |           |           |
| 2025/01/31 |  150.2000 |  151.2000 |  149.2000 |
+------------+-----------+-----------+-----------+
```

#### データ取得

```sql
SELECT
    RATE_DATE,
    TTM,
    TTS,
    TTB
FROM M_EXCHANGE_RATE
WHERE CURRENCY_CODE = @currencyCode
  AND RATE_DATE >= @monthStart
  AND RATE_DATE <= @monthEnd
ORDER BY RATE_DATE
```

#### Excel書式

| セル | 書式 |
|------|------|
| ヘッダ行 | 背景色：濃い青（#4472C4）、フォント色：白、太字 |
| 日付列 | yyyy/MM/dd 形式 |
| レート列 | 小数4桁表示（#,##0.0000）、右揃え |
| データ行 | 交互行の背景色：薄い青（#D9E2F3） |

### 2.8 エラー処理

| エラーケース | 対処 |
|------------|------|
| 対象月のNAVデータが0件 | セクション②～④は「データなし」と表示。セクション①と⑤は出力する |
| リターン計算に必要なNAV（前月末等）が存在しない | 該当リターン欄に「N/A」と表示 |
| 純資産総額の直近12か月データが一部欠落 | 存在する月のみグラフに表示（欠落月は棒が表示されない） |
| 為替レートが0件 | セクション⑤は「データなし」と表示 |
| ファイル出力先に書込権限がない | BusinessException → エラーメッセージ表示 |
